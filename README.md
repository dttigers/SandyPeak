# Sandy Peak Woodcraft

Multi-page static website for Sandy Peak Woodcraft with a gallery lightbox, premade store, and quote request flow.

## Pages
- `index.html` - home + contact form
- `about.html` - studio story and process
- `gallery.html` - gallery with lightbox
- `store.html` - premade products with quote buttons
- `quote.html` - quote cart + request form

## Assets
- `assets/css/style.css`
- `assets/js/site.js`
- `assets/js/quote.js`
- `assets/images/*.webp` (product and shop photography)

## Run locally
Open `index.html` in your browser.

## Quote requests
Quote submissions open a pre-filled email using `mailto:` in `assets/js/quote.js`. Replace the mailto flow with a server endpoint if you want server-side delivery.
